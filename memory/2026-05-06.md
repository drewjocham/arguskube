## Key Learning
- **Module-level refs cause test pollution**: useToast.js had `const toasts = ref([])` at module level, shared across all composable invocations. Moved inside the function.
- **Cache pollution is subtle**: `cachedCallGo` caches by method+args key. When test A succeeds and caches, test B that mocks a rejected fetch still gets cached data. Fix: `invalidateCachePrefix(method)` in beforeEach for all cachedCallGo-using composables.
